# Qkcast

Quickly create a podcast server from a directory of mp3s

## Usage

Once installed use it as
    
    qkcast serve DIR 

Qkcast will create an RSS with all the mp3s in DIR and start serving it.

You can optionally pass `--ip` and `--port`
